# Kettu Plugins Repository

This repository hosts Kettu/Vendetta-style plugins.

## Included Plugins
- `quoter`

## GitHub Pages Install URL

After pushing this repo and enabling GitHub Pages (branch: `main`, folder: `/`), install links are:

```text
https://<your-user>.github.io/<your-repo>/quoter/
```

Use that URL in `Kettu -> Plugins -> Install a plugin`.

## Raw GitHub Install URL (alternative)

```text
https://raw.githubusercontent.com/<your-user>/<your-repo>/main/quoter/
```

Kettu resolves:

- `<plugin-url>/manifest.json`
- `<plugin-url>/index.js`
